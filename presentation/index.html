Middleware «has.university»

Пользователи без указанного университета перенаправляются на редактирование профиля перед доступом к Community

08

Технические решения

Проблема Решение
Дублирование CRUD-кода Trait ManagesResource — единая логика для создания, обновления, удаления и работы с изображениями
Файлы пропадают на Render Supabase Storage через S3-совместимый API. Конфигурируемый диск через FILESYSTEM_DISK
Кэш Windows портит Collections Redis для сессий и кэша. Fallback на DB при недоступности
Mixed Content на Render TrustProxies + ASSET_URL для корректной генерации HTTPS-ссылок
Бесплатный деплой Render.com + Docker + Blueprint. Миграции при старте контейнера
